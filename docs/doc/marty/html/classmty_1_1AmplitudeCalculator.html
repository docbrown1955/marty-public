Parameters

pMomentum to simplify
mMass of the particle
alphaIndex of the external spinor
signTrue if the particle is a particle, false if it is an anti-particle.
Returns
1 if the Dirac equation has been applied.
0 if the momentum has been found in the \( \slashed{p} \) form but could not be simplified (requires Fierz identity).
-1 else.
