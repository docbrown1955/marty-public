Detailed Description

Class containing all model building features of MARTY.

This class inherits ModelData that can store all the data concerning a model. ModelBuilder implements all modifications capabilities of MARTY. Gauge and flavor symmetry breaking, particle replacement / rotation, mass diagonalization, renaming etc. It is further inherited by the Model class that implements all calculation features of MARTY.
